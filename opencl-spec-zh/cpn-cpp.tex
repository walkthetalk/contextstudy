\chapter{OpenCL C++ 外覆 API}

OpenCL C++ 外覆 API 為\cnglo{platform}和運行時 API 提供了 C++ 接口。
他構建於 OpenCL 1.2 C API （包括\cnglo{platform}和運行時）之上，
而不是要取代 C API。
{\ftEmp{要求}}所有 C++ 外覆 API 的實作都要調用底層的 C API，
並假定 C API 是符合 OpenCL 1.2 規範的。

細節請參閱《OpenCL C++ 外覆 API 規範》。
可由如下網址得到：\from[khronosRegistryCL]。
